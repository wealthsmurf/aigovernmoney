# **App Name**: Youth Housing Navigator

## Core Features:

- Eligibility Checker: Determine user eligibility for youth housing subsidies based on age, income, location, and housing type. The tool will compare the user input to the data to output if the user is most likely eligible for support.
- Subsidy Details: Provide the details of subsidies and supporting monetary amount based on user input.
- Call-to-Action: Direct users to application guidelines and relevant application websites.
- Blog Engine: Render markdown files for the blog, using H2 tags for titles, lists, and tips.

## Style Guidelines:

- Primary color: Deep teal (#008080) to convey trust and stability in financial matters.
- Background color: Light teal (#E0F8F7), a very light tint of the primary color to create a clean backdrop.
- Accent color: Yellow-orange (#FFB347) to highlight calls to action.
- Headline font: 'Space Grotesk', a proportional sans-serif font for headlines; body font: 'Inter', a grotesque-style sans-serif
- Use clean, minimalist icons related to housing and finance.
- A clean, card-based layout will provide information to the user
- Subtle animations on button presses, to provide an interactive and playful experience.
